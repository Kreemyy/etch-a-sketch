# Etch A Sketch

A browser-based sketch pad built as part of **The Odin Project** foundations curriculum. You can draw by dragging across the grid, choose custom colors, generate random colors, clear the sketch pad, and resize the grid dynamically.

## Live Demo

[Live Preview](https://kreemyy.github.io/etch-a-sketch/)

## Features

- Dynamic grid generation with JavaScript
- Adjustable grid size (16×16 to 100×100)
- Custom color picker
- Random color mode
- Clear sketch board
- Click and drag drawing effect
- Responsive grid layout using CSS flexbox

## Built With

- HTML
- CSS
- JavaScript

## Implementation Details

The sketch grid is generated dynamically with JavaScript based on the selected grid size. Grid cells are arranged using **CSS flexbox** and **flex-wrap**, allowing the cells to automatically wrap and form a square drawing area.
